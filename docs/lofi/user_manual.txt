index             START SIDAN
---------------------------------------------------------------------------------------------------------------------
till vänster visas text om användaren. denna text läses från config filen. 
till häger visas det senaste projketet med en länk till projektet.  






list              LIST SIDAN 
---------------------------------------------------------------------------------------------------------------------
listar alla projekt. projekten är indelade i div taggar som alla är klickbara (inom <a>) och leder till project sidan 
för detta projekt. 





project           PROJEKT SIDAN 
---------------------------------------------------------------------------------------------------------------------
visar titel, namn och datum samt tekniker som använts. 
text om projektet som kan innehålla html taggar tex för rubriker, bilder och videos. 






techniques        TEKNIK SIDAN 
---------------------------------------------------------------------------------------------------------------------
innehåller en stor och tydlig input där man kan skriva fri sökning och en klick knapp för att utföra sökningen. 
under detta finns det kryssrutor för att välja vissa tekniker. namnet vid sidan av klickrutan är klickbart och 
listar i så fall alla projekt som är knutna till en viss teknik. visas på samma sätt som LIST SIDAN i div taggar.





header 
---------------------------------------------------------------------------------------------------------------------
namn och efternamn läses från config filen 
till höger finns en meny med [start] [projekt] och [sök] som leder till sidorna index, project och tecqniques. 


footer 
---------------------------------------------------------------------------------------------------------------------
kontakt information som läses från config filen 
och till höger statisk information om systemet.
